Default URL Setup Guide
  1. Open Koyfin and navigate to your Whinfell Transmission Map workspace.
  2. Copy the full URL (starts with https://app.koyfin.com/...).
  3. Paste into Default Koyfin Whinfell Workspace URL at top of page.
  4. Open Barchart to your preferred futures / basis view (leadership, CL, ES, etc.).
  5. Copy that URL and paste into Default Barchart Futures / Basis Page URL.
  6. Click Save Settings — done. Settings persist in this browser via localStorage.
  7. Each morning: open this HTML file → click ↻ Refresh All Panes for latest data.

Auto-refresh (optional): Enable the 5-minute checkbox to reload both panes automatically. Works when sites allow iframe embed; otherwise use Open ↗ tabs.
