# How to Use — C2 Whinfell Credit Score Fallback Dashboard

**Deliverable:** `C2_Whinfell_Credit_Score_Fallback.xlsx`  
**Version:** 0.1 (Approved June 26, 2026)

## When to Use

Open this workbook when Comet or Koyfin is slow, unavailable, or you need to manually verify the Whinfell Credit Confirmation Score.

## Quick Start (< 2 minutes)

1. Open **Market Inputs** tab — enter latest 1D% and 5D% for HYG, LQD, XLF, SPY, IBIT
2. Open **Dashboard** tab — set each component **Signal** (dropdown) per C1 logic
3. Read **FINAL SCORE** (cell E17) and interpretation row below
4. Check **Basis Trade Readiness** for Client Basis vs Outright Basis guidance

## Tabs

| Tab | Purpose |
|-----|---------|
| Dashboard | Score engine, interpretation, basis readiness |
| Market Inputs | Observable return data (blue cells = user input) |
| Instructions | Full signal guide and formula reference |
