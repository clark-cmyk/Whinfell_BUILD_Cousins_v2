      Authoritative command bar (2.2): After Import Parquet Hydration or WTM EXPORT v2.1 import, the top bar shows pipeline-trusted Whinfell Score, SQ3, Gate, Gross Risk, and Freshness. A blue Pipeline badge confirms authority. Manual intake edits switch to amber Override until you re-import.

      Freshness indicators: Colored dot + label (Fresh <4h · Aging 4–24h · Stale >24h) with source channel, snapshot ID, and data as-of under the chip. Data Provenance panel in the left rail carries full lineage detail.

      Dual-track flow: Enter Global intake (left) from Koyfin / Prompt A. Enter China SQ3 dimensions or import CHINA POLICY EXPORT v1.0. Command bar shows both scores; gate considers Global for BTC access and SQ3 for China caution.

      Track toggle: Both · Global · China filters visible clusters without hiding intake fields.

      Daily CSV chain (Comet runbook): python3 run_csv_download.py daily --operator desk --window 24h --hydrate-output data/hydration/latest.json — copies Downloads → staged_raw/ (never moves), writes .meta.json sidecars, quarantines bad files, emits manifests, then collect → Parquet and hydrate. Manual steps: init · stage · collect · hydrate. Alt: python -m whinfell_pipeline.ingest --staged.

      Daily hydration (Comet runbook): python3 run_csv_download.py daily --hydrate-output data/hydration/latest.json then Import Latest Hydration Bundle (select that file). Status chip under the button confirms last import. Auto-populates Global intake, China policy, BTC bias + L3 basis when present, and tracer suggestions — never written to the horizon matrix until you Accept.
